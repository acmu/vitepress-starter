### dropdown 组件

使用 `placement="bottom"` 可以让下面`dropdown menu`的内容居中展示，这个的默认值是`bottom-end`

## $attrs

[vue中的$attrs是什么？](https://www.jianshu.com/p/991c66ec9a62)

父组件写在了子组件的 props 上，但子组件并没有接收 props，这时就会出现在 $attrs 中，在子组件中常用 `v-bind="$attrs"`透传属性
