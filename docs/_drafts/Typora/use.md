## 常用的



| 快捷键 | 意义 |
| :-- | :-- |
| opt cmd c | 代码块 |
| opt cmd t | 表格 |
| cmd n | n 级标题 |
| cmd + enter | 向下加一行（在表格中） |
| cmd + shift + backspace | 删除当前行（在表格中） |
| $$ + enter | [使用 latex 数学语法](https://support.typora.io/Math/) |
| `- [ ] task list` | 这样输入可以出现 task list |
| 输入`[toc]` 回车 | 展示出内容表格 |
| $a\times b$ |  |
| $\frac{a}{b}$ |  |
| cmd + shift + L | 切换展示侧边栏 |
